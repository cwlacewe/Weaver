Weaver_SV
===============

###############
INSTALL
###############

Bamtools (https://github.com/pezmaster31/bamtools) libraries are needed

cd bamtools
mkdir build
cd build
cmake ..
make

Parallel::ForkManager perl packages are needed

under Weaver_SV
cmake CMakeLists.txt
make


###############
INPUT
###############

BAM file from BWA


###############
OUTPUT
###############

VCF file for SV
